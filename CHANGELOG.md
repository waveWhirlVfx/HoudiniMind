# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-28
### Added
- Initial public release of HoudiniMind.
- Multi-agent framework for driving SideFX Houdini 21 via a curated, schema-validated tool API.
- Local-first execution using Ollama for privacy and performance.
- Hierarchical planning and verification pipeline (Plan → Act → Observe → Verify).
- Vision-guided self-correction and multi-view semantic scoring.
- Deep RAG integration with 700+ SOP node docs and FX recipes.
- Model Context Protocol (MCP) server for external editor integration (Cursor, etc.).
- Comprehensive "Zero-Warning" tool API with built-in schema validation.

# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct. Please read the full text at the link above.

## Scope

The Code of Conduct applies to all project spaces: the GitHub repository,
issues, pull requests, discussions, and any communication channel run on
behalf of the project.

## Reporting

Report unacceptable behaviour to the maintainers at **vashistanshul.7@gmail.com**.
Reports are reviewed within 72 hours and handled confidentially. Maintainers
who violate the Code of Conduct may face temporary or permanent removal from
the project at the discretion of the remaining maintainers.

## Enforcement

Maintainers follow the Contributor Covenant's
[Enforcement Ladder](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines):
correction → warning → temporary ban → permanent ban.
